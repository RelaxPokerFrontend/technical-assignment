# C++ Frontend UI Signal Test
This is a small C++ assignment intended for frontend-oriented developers (UI, client logic, event-driven systems).

The goal is to implement:
- A simple UI event queue
- A worker thread waiting for signals/events
- A minimal UI component system
- A demo showing how UI updates react to events

## Development Environment
**The candidate is free to choose any development environment.**

You may use:
- Visual Studio
- CLion
- VSCode
- Xcode
- Linux terminal
- Windows terminal
- Any C++17-compatible compiler (MSVC, Clang, GCC)

No external libraries are required.
